css 三大特性:
  • 继承性
  • 层叠性
  • 优先级

继承性

css属性继承, 只有以 color, font, line, text 开头的属性才可以被继承
不仅儿子可以继承, 后代都可以继承属性
特殊性:

  1. a标签的颜色和下划线不能继承
  2. h标签文字大小不能继承

层叠性

css 处理冲突的一种能力, 同一属性多次设置, 只有一种起作用 css 层叠式样式表

优先级

梦里花 文件内 css 设置属性

欧若拉 标签内css 设置属性

淋雨一直走    外部css文件通过 id设置属性, 优先级大于标签

!important : 提高css修饰的属性优先级, 注意点:
  • !important 只能用于直接选中, 不能用于间接选中
  • 通配符选中的标签也是直接选中的
  • 只能提升被指定的属性优先级, 对其他属性无作用
  • 必须写在属性值的分号前面
  • !important 前面的! 不能省略

权重

当多个选择器混合使用, 选中同一个标签时, 权重判断规则:
  1. 先计算id数量, id多的优先级大
  2. 如果id数量相同, 类名多的优先级大
  3. 如果id和类名数量相同, 标签名多的优先级大
  4. 如果id,类名,标签名数量都相同, 谁在后面谁的优先级大
